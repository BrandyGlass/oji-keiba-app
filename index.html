おじ.exe（競馬プロジェクト）

目的：迷わない／感情で買わない／決めた手順通りに動く
STEP0〜STEP4 実装済み（次：STEP5〜ログ強化）。

複勝1点 1R 500円 最大3レース/日 得意条件：中山ダ1800 / 東京ダ1600 東京芝1400：一時停止

STEP0｜前提確認（10秒）

Q1：今日は最大3レースまで、というルールを守れる？

No が出たら、その時点で終了（今日は見送り）。

Q2：このレースは500円だけで買う？

「500円だけ」が守れないなら、このレースは見送り。

STEP1｜対象レース判定（即断）

Q3：このレースは重賞（G1〜G3）？

重賞は買う対象に含める（ただしSTEP0の資金ルールは絶対）。

Q4：得意条件に当てはまる？

東京芝1400は一時停止（選択肢に存在しない）。

✅ 対象レース：OK

見送りは正解ルート。迷ったら止める。

STEP2｜候補抽出（番号で最大3頭）

この工程では本命を決めない。
「変な負け方をしにくそう」な馬を番号で最大3頭まで置くだけ。人気・オッズは見ない。

Q2：候補馬の番号（最大3頭）

ルール：数字のみ／同じ番号は不可。
馬①が書けない＝候補が出ない → 見送りが合理的。

STEP3-①｜コース × 距離（最優先）

選択肢で評価する（迷ったら下げる）。
◎2 / ◯1 / △0 / ×-2（※点数化はSTEP4でやる）

書けない（選べない）が出たら見送りでOK。

STEP3-②｜流れ × 脚質（負け筋の少なさ）

勝ち方ではなく「どう負けるか」を先に決める。迷ったら下げる。
◎2 / ◯1 / △0 / ×-2

負け筋が2つ以上浮かぶ感じなら△か×で良い（それが正直）。

STEP3-③｜騎手（減点専用）

ここは加点しない。「壊さないか」だけ。
OK 0 / 不安 -0.5 / NG -1

不明は「不安」扱いにしてOK（迷ったらブレーキ）。

STEP4｜点数化（機械的に本命を決める）

合計点 = ① + ② + ③。トップが本命。
同点：① → ② → ③ → それでも同点なら見送り。
さらに、トップが 1点未満 なら見送り推奨。

STEP5〜は次で「0番チェック→結論→ログ強化」を入れる。

見送り
